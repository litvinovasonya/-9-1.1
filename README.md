В конце работы вывела общее количество студентов и все полученные зависимости для передачи работы участнику 2.

Во второй части модуля я связала свою работу с результатами участника 1: взяла готовый список и добавила два поля с лучшими и худшими студентами. Елизарова Мария. Я написала модуль для подробного анализа 20 лучших студентов и создания рекомендаций для попадания в этот топ (recommendations.py). В первой части модуля я реализовала функцию для сбора статистики по ключевым показателям топ-20 студентов: часы учёбы, сна, посещаемость и итоговый балл. В конце работы я вывела итоговые рекомендации и оформила результаты в виде словаря для дальнейшего использования в проекте, добавив свою часть в результат работы участника 3. Также я написала модуль, который будет запускать пользователь (main.py), включающий в себя общий вывод всей проделанной работы (загрузка данных и анализ зависимостей, поиск лучших и худших студентов, анализ привычек и рекомендации для попадания в топ-20 студентов).
